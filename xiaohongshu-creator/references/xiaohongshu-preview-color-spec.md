# 小红书笔记预览模板 - 配色方案与使用指南

## 配色理念

Vital Geometry 配色系统基于**温暖的米色中性基底**，点缀**精心选择的强调色**，营造出既专业又充满活力的视觉体验。

**核心原则：**
- 大面积使用米白/灰色调，营造舒适感
- 少量使用珊瑚红和琥珀黄作为强调，增加活力
- 严格控制对比度，确保可读性
- 所有颜色都带有温暖倾向，避免冷峻感

## 2. 渐变色方案

### 2.1 顶部彩条渐变

```css
/* 标准卡片顶部彩条 */
background: linear-gradient(90deg, #1a1a1a 0%, #3a3a3a 100%);

/* 推广卡片顶部彩条 */
background: linear-gradient(90deg, #ff6b6b 0%, #ffc107 100%);
```

**使用场景：**
- 标准卡片：深灰渐变，稳重专业
- 推广卡片：红黄渐变，醒目活泼

### 2.2 背景光晕渐变

```css
background-image:
    radial-gradient(circle at 20% 30%, rgba(255, 107, 107, 0.03) 0%, transparent 50%),
    radial-gradient(circle at 80% 70%, rgba(255, 193, 7, 0.02) 0%, transparent 50%);
```

**设计要点：**
- 多层径向渐变叠加
- 位置分散，避免集中在中心
- 极低透明度，仅增加氛围感

### 2.3 数据框角装饰渐变

```css
background: linear-gradient(135deg, transparent 50%, rgba(255, 193, 7, 0.05) 50%);
```

**设计要点：**
- 对角线渐变
- 一半透明，一半淡琥珀黄
- 仅用于左上角 60px × 60px 区域

### 2.4 推广卡片强调框渐变

```css
background: linear-gradient(135deg, rgba(255, 107, 107, 0.08) 0%, rgba(255, 193, 7, 0.08) 100%);
```

**设计要点：**
- 对角线渐变
- 从珊瑚红到琥珀黄
- 极低透明度，保持可读性

## 3. 颜色使用规则

### 3.1 主次层级

**第一层级（主色）：**
- 背景：#fafaf8（米白）
- 文字：#1a1a1a（近黑）
- 占比：70-80%

**第二层级（辅助色）：**
- 文字：#555、#666、#999（灰度）
- 背景：#f8f8f6、#f5f3f0（浅灰）
- 占比：15-20%

**第三层级（强调色）：**
- 强调：#ff6b6b（珊瑚红）
- 点缀：#ffc107（琥珀黄）
- 占比：5-10%

### 3.2 对比度要求

**正文文字（WCAG AA 标准）：**
- #333 on #fff：对比度 12.6:1 ✅
- #555 on #fafaf8：对比度 7.5:1 ✅
- #666 on #fafaf8：对比度 5.7:1 ✅

**强调文字：**
- #ff6b6b on #fafaf8：对比度 4.8:1 ✅
- #ffc107 on #fafaf8：对比度 4.2:1 ⚠️（谨慎使用）

**辅助文字：**
- #999 on #fff：对比度 4.5:1 ✅（最小可接受）

### 3.3 使用场景指南

#### 珊瑚红 (#ff6b6b) 使用场景

**✅ 适合：**
- 标题装饰线（30px × 3px）
- 列表项圆点（8px）
- 强调文字（关键词）
- 数据框中的数值强调
- 装饰性光晕

**❌ 避免：**
- 大面积背景
- 正文文字
- 超过 3 处强调（避免过度使用）

#### 琥珀黄 (#ffc107) 使用场景

**✅ 适合：**
- 高亮圆点组（3-4 个小圆点）
- 数据框角装饰（渐变）
- 背景光晕
- 与珊瑚红搭配的渐变

**❌ 避免：**
- 正文文字（对比度不足）
- 大面积背景
- 作为唯一强调色

#### 蓝紫色 (#667eea) 使用场景

**✅ 适合：**
- 标签云背景
- CTA 按钮
- 链接文字

**❌ 避免：**
- 卡片内主要元素（保持风格统一）
- 过度使用（打破配色平衡）

## 4. 特殊场景配色

### 4.1 风险提示框

```css
background: #fff3cd;
border: 1px solid #ffc107;
color: #856404;
```

**配色逻辑：**
- 背景色：浅黄（警示色）
- 边框色：琥珀黄（与背景同色系）
- 文字色：深棕黄（确保对比度）

### 4.2 选中状态

**标题项选中：**
```css
background: #fff;
border-color: #e8e8e8;
color: #1a1a1a;
```

**圆点指示器选中：**
```css
background: #1a1a1a;
```

### 4.3 悬停状态

```css
background: #f5f5f5;
```

**设计要点：**
- 仅改变背景色，不改变文字颜色
- 保持轻微变化，不过度强调

## 5. 装饰元素配色

### 5.1 点阵网格

```css
.dot {
    background: #ddd;
}
```

**透明度变化：**
- 默认：opacity: 1
- 装饰用：opacity: 0.2-0.3
- 白色卡片上：opacity: 0.3-0.4

### 5.2 波浪线

```css
background: repeating-linear-gradient(
    90deg,
    transparent,
    transparent 4px,
    rgba(0, 0, 0, 0.03) 4px,
    rgba(0, 0, 0, 0.03) 8px
);
```

**设计要点：**
- 极低透明度（0.03）
- 透明与实线交替

### 5.3 同心圆

```css
circle {
    stroke: rgba(0,0,0,0.03);  /* 外层 */
}
circle {
    stroke: rgba(0,0,0,0.04);  /* 中层 */
}
circle {
    stroke: rgba(0,0,0,0.05);  /* 内层 */
}
```

**设计要点：**
- 透明度递增
- 保持极低，不抢视线

## 7. 暗色模式（可选）

如需支持暗色模式，建议使用以下配色：

```css
@media (prefers-color-scheme: dark) {
    /* 背景 */
    --bg-primary: #1a1a1a;
    --bg-secondary: #2d2d2d;
    --bg-tertiary: #3a3a3a;

    /* 文字 */
    --text-primary: #fafaf8;
    --text-secondary: #ccc;
    --text-tertiary: #999;

    /* 强调色保持不变 */
    --accent-red: #ff6b6b;
    --accent-yellow: #ffc107;
}
```

## 8. 配色最佳实践

### 8.1 使用原则

1. **70-20-10 法则**：
   - 70% 中性色（米白、灰）
   - 20% 辅助色（灰度文字）
   - 10% 强调色（珊瑚红、琥珀黄）

2. **限制强调色数量**：
   - 每张卡片最多 2-3 处珊瑚红
   - 每张卡片最多 1-2 处琥珀黄

3. **保持温暖感**：
   - 所有中性色都带有暖色调
   - 避免使用纯白 (#fff) 以外的冷白色

4. **对比度优先**：
   - 确保所有文字与背景对比度 ≥ 4.5:1
   - 正文文字建议对比度 ≥ 7:1

### 8.2 常见错误

**❌ 错误 1：过度使用强调色**
```css
/* 不好的做法 */
.card-title { color: #ff6b6b; }
.card-text { color: #ff6b6b; }
.card-link { color: #ff6b6b; }
```

**✅ 正确做法：**
```css
.card-title { color: #1a1a1a; }
.card-title::after { background: #ff6b6b; }  /* 仅用于装饰 */
```

**❌ 错误 2：对比度不足**
```css
/* 不好的做法 */
.text { color: #ccc; }  /* 在浅灰背景上对比度不足 */
```

**✅ 正确做法：**
```css
.text { color: #666; }  /* 确保可读性 */
```

**❌ 错误 3：配色混乱**
```css
/* 不好的做法 */
.element1 { color: #ff6b6b; }
.element2 { color: #00ff00; }  /* 不协调的绿色 */
.element3 { color: #0000ff; }  /* 不协调的蓝色 */
```

**✅ 正确做法：**
```css
.element1 { color: #ff6b6b; }  /* 珊瑚红强调 */
.element2 { color: #ffc107; }  /* 琥珀黄点缀 */
.element3 { color: #667eea; }  /* 蓝紫标签 */
/* 保持配色系统的一致性 */
```

## 9. 颜色变量定义

建议使用 CSS 变量管理颜色：

```css
:root {
    /* 背景色 */
    --bg-primary: #fafaf8;
    --bg-white: #fff;
    --bg-section: #f5f3f0;
    --bg-box: #f8f8f6;
    --bg-hover: #f5f5f5;

    /* 文字色 */
    --text-primary: #1a1a1a;
    --text-secondary: #333;
    --text-tertiary: #555;
    --text-quaternary: #666;
    --text-light: #999;
    --text-lighter: #ccc;

    /* 强调色 */
    --accent-red: #ff6b6b;
    --accent-yellow: #ffc107;
    --accent-blue: #667eea;

    /* 边框色 */
    --border-light: #e8e8e8;
    --border-medium: #ddd;
    --border-dark: #f0f0f0;

    /* 装饰色 */
    --decoration-dot: #ddd;
    --decoration-line: rgba(0,0,0,0.03);
}
```

## 10. 配色检查清单

使用前检查：

- [ ] 所有文字对比度 ≥ 4.5:1
- [ ] 正文文字对比度 ≥ 7:1
- [ ] 每张卡片强调色 ≤ 3 处
- [ ] 装饰元素透明度 ≤ 0.3
- [ ] 背景色使用 70% 以上
- [ ] 强调色使用 ≤ 10%
- [ ] 没有引入新的不协调颜色
- [ ] 保持温暖色调倾向
- [ ] 渐变色使用合理
- [ ] 特殊场景（警告、选中）配色清晰
