# 账号风格选择配置

本文档定义可用的账号风格及其使用场景。

---

## 可用账号风格

### 1. 叮叮猫的财经笔记

**风格文件**：`account-styles/dingdingmao.md`

**账号类型**：小红书KOS账号

**适用场景**：
- 金融知识科普（股票指标、ETF、基金选择）
- 投资理财教程
- 理财小白友好内容
- 需要亲和力的话题

**目标受众**：投资新手、理财小白

**核心特点**：口语化、生活化、伙伴式、知识分享

**引流方式**：软性引导，私信领取资料包

**典型表达**："宝子们""速存""划重点""一键抱走"

### 4. 证券人排雷哥笔记

## 风格选择指南

### 根据内容类型选择

| 内容类型 | 推荐账号风格 | 账号类型 |
|---------|------------|---------|
| 金融知识入门科普 | 叮叮猫的财经笔记 | KOS |
| 资产配置教育 | 小红书主号-投教笔记 | 官方 |
| 开户佣金推广 | 小红书主号-广告笔记 | 官方 |
| 系统化工具讲解 | 证券人排雷哥笔记 | KOS |
| 市场热点解读 | 申万宏源乐乐笔记 | KOS |
| 深度市场分析 | 仿中金公众号文风 | 公众号 |

### 根据账号身份选择

**官方账号/员工账号** →
- 小红书主号-投教笔记（偏知识教育）
- 小红书主号-广告笔记（偏直接推广）

**KOS知识分享账号** →
- 叮叮猫的财经笔记（偏小白友好）
- 证券人排雷哥笔记（偏系统讲解）
- 申万宏源乐乐笔记（偏热点解读）

**公众号/深度内容** →
- 仿中金公众号文风（偏理性分析）

## 默认风格设置

### 如果用户未指定风格

**默认使用**：小红书主号-投教笔记（申万宏源证券财富管理官方）

**原因**：
- 作为财经证券领域自媒体
- 官方/员工账号需要体现专业性
- 平衡了知识普及与服务推广
- 最适合通用场景
