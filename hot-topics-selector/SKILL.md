# 财经热点选题工具

## 任务目标

本 Skill 用于：基于真实财经热点资讯，智能筛选适合内容创作的选题

核心能力包含：
- **热点抓取**：实时获取 8+ 平台财经热点
- **需求理解**：理解创作意图和目标受众
- **智能筛选**：多维度评估选题价值
- **创作建议**：提供结构化的创作大纲
- **质量评估**：评估选题的爆款潜力

触发条件：
- 用户要求选题、寻找创作灵感
- 获取热点资讯、追踪市场动态
- 准备创作财经内容

## 核心工作流

```
用户需求 → Phase 0(热点抓取) → Phase 1(需求理解) →
Phase 2(选题分析) → Phase 3(创作建议✋) → Phase 4(质量评估✋)
```

**关键特点**：
- 基于真实数据，非虚构热点
- 结构化思考，系统化评估
- 逐步确认，用户参与决策

## Phase 0: 热点抓取

**任务**：获取实时财经热点数据

**执行要求**：
1. 调用 `scripts/fetch_hot_topics.py` 抓取热点
2. 支持指定平台和关键词过滤
3. 生成数据摘要

**平台支持（8个）**：
- 财经专业：财联社、36氪、格隆汇
- 综合资讯：今日头条、百度热搜、微博
- 短视频：抖音
- 知识问答：知乎

**使用方法**：

```bash
# 获取财经平台热点
python3 scripts/fetch_hot_topics.py --finance

# 获取所有平台热点
python3 scripts/fetch_hot_topics.py

# 关键词筛选
python3 scripts/fetch_hot_topics.py --keywords "AI,芯片,新能源"

# 指定平台
python3 scripts/fetch_hot_topics.py --platforms cls-hot,gelonghui
```

**输出格式**：
```json
{
  "fetch_time": "2026-02-13T19:36:24+08:00",
  "platforms_count": 3,
  "total_items": 28,
  "data": {
    "cls-hot": {
      "name": "财联社热门",
      "items": [...],
      "count": 13
    }
  }
}
```

## Phase 1: 需求理解与重构 ✋ 必须确认

**任务**：深度分析用户创作需求

**执行原则**：⚠️ **必须逐步执行，每步完成后等待用户确认**

### Step 1.1: 创作场景识别

**任务**：理解创作场景和目标

**分析维度**：
- **创作平台**：小红书 / 微信公众号 / 抖音 / 其他
- **内容类型**：科普 / 分析 / 观点 / 教程
- **目标受众**：新手 / 进阶 / 专业
- **转化目标**：涨粉 / 引流 / 品牌 / 教育

**输出格式**：
```markdown
## 创作场景分析

**创作平台**：<平台名称>

**内容类型**：<类型描述>

**目标受众**：<受众特征>

**转化目标**：<期望效果>

请确认是否准确？
```

### Step 1.2: 热点匹配策略

**任务**：确定热点筛选策略

**策略选项**：
- **高热度策略**：选择热度最高的选题（适合涨粉）
- **高相关策略**：选择最匹配目标受众的选题（适合引流）
- **差异化策略**：选择冷门但有深度的选题（适合品牌）
- **时效性策略**：选择最新鲜的选题（适合快讯）

**输出格式**：
```markdown
## 热点匹配策略

**推荐策略**：<策略名称>

**选择理由**：<为什么这个策略适合当前场景>

**筛选标准**：
- 热度要求：≥ X 分
- 相关性：≥ Y 分
- 时效性：< Z 小时

是否采用此策略？或提出修改意见。
```

## Phase 2: 选题分析 ⭐ 核心阶段

**任务**：智能分析热点，推荐优质选题

### Step 2.1: 多维度评分

**评分体系（0-100分）**：

1. **热度分（40%）**
   - 排名权重：排名越靠前分数越高
   - 平台权重：财经平台额外加成
   - 频次权重：多平台出现额外加成

2. **相关性分（30%）**
   - 财经关键词匹配度
   - 目标受众匹配度
   - 创作场景匹配度

3. **创作价值分（30%）**
   - 内容深度潜力
   - 差异化空间
   - 时效性评估

### Step 2.2: 选题推荐

**任务**：生成 Top N 选题推荐

**执行要求**：
- 调用 `scripts/analyze_topics.py` 分析数据
- 根据用户策略筛选选题
- 生成详细报告

**使用方法**：

```bash
# 分析选题
python3 scripts/analyze_topics.py --input /tmp/hot_topics_xxx.json --top 10

# 输出格式
python3 scripts/analyze_topics.py --input /tmp/hot_topics_xxx.json --format json
```

**输出格式**：
```markdown
## 🔥 Top 10 选题

### 1. [选题标题]

**综合评分：** 89.5/100

**评分详情：**
- 热度：100/100
- 相关性：80/100
- 创作价值：85/100

**来源：** 财联社 | 排名: #2

**推荐理由：** 🔥 强烈推荐：高热度、强相关性、优质创作角度

**创作角度：**
- 角度1：...
- 角度2：...
- 角度3：...

**目标受众：** ...

**时效性：** ⏰ 高（建议24小时内发布）

---
```

### Step 3.2: 创作要点提示

**任务**：提供创作注意事项

**要点包括**：

- **数据支撑**：需要准备的数据
- **案例素材**：建议使用的案例

- **合规要点**：必须包含的风险提示

## Phase 4: 质量评估 ✋ 必须确认

**任务**：评估选题的创作质量

### Step 4.1: 选题质量检查

**检查清单**：

**✅ 内容价值**
- [ ] 有明确的核心观点
- [ ] 对目标受众有价值
- [ ] 内容深度足够
- [ ] 时效性符合预期

**✅ 差异化**
- [ ] 与竞品内容有区别
- [ ] 有独特视角或分析
- [ ] 不是简单的信息搬运

**✅ 可执行性**
- [ ] 素材充足（数据、案例）
- [ ] 创作难度适中
- [ ] 时效性充足

**✅ 合规性**
- [ ] 不触碰敏感话题
- [ ] 不传播虚假信息
- [ ] 可以加入风险提示

### Step 4.2: 爆款潜力评估（小红书专用）

**评估维度（0-100分）**：

1. **标题吸引力（25%）**
   - 是否有数字/对比/冲突
   - 是否引发好奇心
   - 是否与受众相关

2. **内容价值（25%）**
   - 是否解决实际问题
   - 是否有干货输出
   - 是否容易理解

3. **情感共鸣（25%）**
   - 是否触动痛点
   - 是否有代入感
   - 是否有传播动力

4. **视觉友好度（25%）**
   - 是否适合图文呈现
   - 是否有视觉化空间
   - 是否需要数据图表

**评分标准**：
- 90-100分：🔥 极具爆款潜力
- 80-89分：✅ 优质选题
- 70-79分：💡 可以尝试
- <70分：⚠️ 建议更换选题

## 常见问题

**Q1: 热点抓取失败怎么办？**
A: 检查网络连接，确认 API 可访问，必要时配置代理

**Q2: 如何判断选题质量？**
A: 综合评分 > 80 分为优质选题，70-80 分为合格选题

**Q3: 不同平台如何选择策略？**
A: 小红书→高相关策略；年年有鱼→时效性策略；品牌内容→差异化策略

**Q4: 如何避免同质化？**
A: 找独特角度、深入分析、结合专业知识、增加个人观点
