# 财经热点选题工具

## 任务目标

- 本 Skill 用于：基于真实财经热点资讯，智能筛选适合内容创作的选题
- 能力包含：热点抓取、关键词筛选、选题评分、选题推荐
- 触发条件：用户要求选题、寻找创作灵感、获取热点资讯

## 前置准备

- 依赖说明：scripts 脚本所需的依赖包及版本
  ```
  requests>=2.22.0
  pytz>=2021.1
  ```

## 核心功能

### 1. 热点抓取

**支持平台：**
- 财经专业：财联社热门、36氪、格隆汇
- 综合资讯：今日头条、百度热搜、微博
- 短视频：抖音
- 知识问答：知乎

**数据维度：**
- 标题、排名、热度
- URL 链接、移动端链接
- 出现频次、时间分布

### 2. 智能筛选

**筛选维度：**
- 关键词匹配度
- 热度指数（排名 + 频次）
- 财经相关性
- 创作价值评估

### 3. 选题评分

**评分体系（0-100分）：**
- 热度分（40%）：排名、频次、传播度
- 相关性（30%）：财经关联度、目标受众匹配
- 创作价值（30%）：内容深度、时效性、差异化

### 4. 选题推荐

**输出内容：**
- 热点标题 + 来源
- 推荐理由
- 创作角度建议
- 目标受众

## 操作步骤

### 标准流程

1. **热点抓取**
   - 调用 `scripts/fetch_hot_topics.py` 获取实时热点
   - 支持指定平台和关键词过滤
   - 输出格式：JSON 文件

2. **选题分析**
   - 调用 `scripts/analyze_topics.py` 进行选题评估
   - 计算热度、相关性、创作价值
   - 输出推荐选题列表

3. **生成报告**
   - 生成选题推荐报告
   - 包含热点详情、评分、创作建议
   - 支持多种格式输出（Markdown/JSON/HTML）

## 使用方法

### 基础用法

```bash
# 获取全部平台热点
python3 scripts/fetch_hot_topics.py

# 获取指定平台热点
python3 scripts/fetch_hot_topics.py --platforms cls-hot,_36kr,gelonghui

# 关键词筛选
python3 scripts/fetch_hot_topics.py --keywords "AI,新能源,芯片"
```

### 高级用法

```bash
# 完整分析流程
python3 scripts/fetch_hot_topics.py
python3 scripts/analyze_topics.py --input /tmp/hot_topics.json

# 生成选题报告
python3 scripts/generate_report.py --top 10 --format markdown
```

## 创作建议

### 选题判断标准

**优先选择：**
- ✅ 热度高且持续性强
- ✅ 与投资/理财直接相关
- ✅ 有明确的创作角度
- ✅ 目标受众清晰

**谨慎选择：**
- ⚠️ 时效性过强（几小时后过期）
- ⚠️ 过于小众，受众有限
- ⚠️ 缺乏深度挖掘空间

**避免选择：**
- ❌ 敏感话题（政策、监管）
- ❌ 虚假信息、谣言
- ❌ 与财经无关的娱乐八卦

### 内容方向

**针对不同平台：**
- 小红书：生活化、科普向、实用技巧
- 年年有鱼：专业分析、投资建议
- 抖音：快节奏、观点鲜明
- 微信公众号：深度解读、案例分析

## 注意事项

### 数据获取

**API 限制：**
- 请求间隔：至少 1 秒
- 失败重试：最多 2 次
- 代理配置：可选

**数据时效：**
- 热点数据实时更新
- 建议定期刷新（1-2小时）
- 避免使用过期数据

### 内容合规

**必须遵守：**
- 不传播虚假信息
- 不触碰敏感话题
- 包含风险提示
- 标注信息来源

## 常见问题

**Q1: 热点抓取失败怎么办？**
A: 检查网络连接，确认 API 可访问，必要时配置代理

**Q2: 如何判断选题质量？**
A: 综合评分 > 80 分为优质选题，70-80 分为合格选题

**Q3: 热点时效性如何把握？**
A: 优先选择热度持续上升的话题，避免即将过期的热点

**Q4: 如何避免同质化？**
A: 找独特角度、深入分析、结合专业知识
