# 图片提示词模板

> 本文档为生成的专属小红书创作Skill提供图片生成提示词模板
> 基于账号风格画像自动生成

---

## 使用方法

根据笔记主题，从模板库中选择合适的提示词模板，填充变量后生成图片。

---

## 模板类型

### 1. 手绘风模板

**适用场景**：知识科普、干货清单、观点输出

**模板结构**：
```
{主体描述}，{风格描述}插画风格，{配色描述}，{构图描述}，{装饰元素}，高清，扁平化，简笔画，手绘质感，{输出比例}
```

**示例**：
```
一个年轻女性在咖啡馆工作，手持笔记本电脑，粉色和米色为主色调，居中构图，右上角有小型装饰图案，无文字，高清，扁平化插画风格，手绘质感，3:4竖版
```

### 2. 实拍风模板

**适用场景**：好物测评、vlog图文、生活分享

**模板结构**：
```
{主体描述}，{场景描述}，{光线描述}，{色调描述}，{构图描述}，{风格描述}，{处理描述}，{输出比例}
```

**示例**：
```
桌上摆放的MacBook笔记本和咖啡杯，旁边有绿色植物，窗边自然光，暖色调，居中构图性冷淡风，轻微过曝，浅灰背景，3:4竖版
```

### 3. 海报风模板

**适用场景**：金句输出、观点表达、清单总结

**模板结构**：
```
{背景描述}，大号{字体描述}居中排列，{副标题描述}，{装饰元素描述}，{配色描述}，{整体风格描述}，{输出比例}
```

**示例**：
```
纯色渐变背景（浅粉到浅蓝），大字"存钱真的不难"居中，手写体字体，左下角有小 money emoji装饰，莫兰迪色系，简约现代风格，3:4竖版
```

### 4. 信息图风模板

**适用场景**：数据展示、教程说明、对比分析

**模板结构**：
```
{主题描述}信息图风格，{布局描述}，{数据展示形式描述}，{配色描述}，{图标风格描述}，{文字比例描述}，{输出比例}
```

**示例**：
```
理财习惯对比信息图，左侧柱状图右侧清单形式，莫兰迪蓝绿色系，简约线性图标，文字占30%，信息图，清晰易读，3:4竖版
```

### 5. 混搭风模板

**适用场景**：种草推荐、好物分享

**模板结构**：
```
{手绘元素描述}叠加在{实拍照片描述}上，{配色描述}，{整体风格描述}，{输出比例}
```

**示例**：
```
手绘风格的花朵和星星装饰叠加在实拍的口红照片上，粉色和金色点缀，梦幻少女风，高清，混搭风格，3:4竖版
```

## 风格后缀

追加在提示词末尾，控制整体风格：

| 风格 | 后缀词 |
|:---|:---|
| 日系 | `--style raw` `ar 3:4` `q 2` |
| 杂志风 | `editorial photo, magazine layout` |
| 手绘质感 | `hand-drawn, illustration style` |
| 扁平插画 | `flat illustration, vector art` |
| 真实照片 | `photorealistic, 8k, detailed` |
| 复古 | `vintage, retro aesthetic` |

**账号风格**：温馨治愈、手绘风、莫兰迪色系

**笔记主题**：职场女性存钱技巧

1. 选择手绘风模板
2. 填充：主体=职场女性+存钱罐，风格=手绘，配色=莫兰迪粉
3. 添加风格后缀：hand-drawn, soft edges

**最终提示词**：
```
一个职场女性手持存钱罐，周围有金币和计算器装饰，手绘插画风格，柔和粉色和米色配色，居中构图，小星星装饰，hand-drawn, soft edges, illustration style, 3:4 vertical
```
