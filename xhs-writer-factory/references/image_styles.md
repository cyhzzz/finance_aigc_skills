# 图片风格预设

> 6种预设风格，可直接使用或作为反推锚点

---

## 6种风格总览

| 风格 | 配色 | 基调 | 适用场景 |
|:---|:---|:---|:---|
| professional | 深蓝+金色 | 专业可信 | 券商开户、费率优惠 |
| fresh | 粉蓝渐变 | 活力亲和 | 新手理财、教育 |
| tech | 深蓝紫+霓虹 | 创新前沿 | 量化交易、ETF |
| warm | 橙黄渐变 | 友好乐观 | 社区运营、长期投资 |
| minimal | 黑白灰 | 简洁现代 | 数据榜单、干货分享 |
| nature | 绿色+大地色 | 有机健康 | 基金定投、稳健理财 |

---

## 封面图要求

- 比例：3:4竖版
- 标题突出显示在中上部
- 对比色提高可读性
- 金币、图表、专业图标

## 正文卡片图要求

- 比例：3:4竖版
- 清晰易读的文字排版
- 充足留白、分区展示

---

## 使用流程

### 用户提供图片
```
提取风格 → 对比6预设找匹配 → 以预设为锚点优化
```

### 用户无法提供图片
```
询问选择预设风格 → 直接使用预设
```

### 询问话术
```
"您希望用什么图片风格？
1. professional（专业深蓝金）
2. fresh（清新粉蓝）
3. tech（科技紫霓虹）
4. warm（温暖橙黄）
5. minimal（简约黑白）
6. nature（自然绿棕）
或者您可以提供参考图片，我来分析风格。"
```
