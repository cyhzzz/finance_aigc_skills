# 微博140字规范

> 本文档是多平台改写流程 Step 6.3 的详细操作手册

---

## 微博输出要求

### 格式要求

| 要求 | 说明 |
|-----|------|
| 字数限制 | ≤140字（含话题标签） |
| 文件格式 | Markdown（.md） |
| 核心观点 | 1句话 |
| 互动引导 | 必须有 |

---

## 微博结构

```markdown
{核心观点（1句话）} {补充说明（1-2句）} {话题标签} {互动引导语}
```

### 字数分配

| 部分 | 字数 | 说明 |
|-----|------|------|
| 核心观点 | 20-30字 | 最核心的判断 |
| 补充说明 | 20-40字 | 支撑或延伸 |
| 话题标签 | 10-30字 | 2-3个话题 |
| 互动引导 | 5-15字 | 可选 |
| **合计** | ≤140字 | 严格控制 |

---

## 微博句式模板

### 核心观点句式

```
{观点}。
例如：
- 市场缩量整固，节前大概率维持震荡
- 这个政策的影响，比表面看起来更深
- XX板块可能是今年最值得关注的方向
```

### 补充说明句式

```
{原因/数据/逻辑}。
例如：
- 成交额跌破万亿，情绪偏谨慎
- 3个信号告诉你为什么
- 值得关注的原因是...
```

### 话题标签

```
#A股# #市场分析# #投资干货#
#基金# #理财# #今日看盘#
#投顾观点# #市场有风险，投资需谨慎#
```

### 互动引导语

```
你怎么看？
你认同吗？
评论区聊聊
你觉得呢？
```

## 微博合规要求

### 字数控制

- 总字数 ≤ 140字
- 含链接时：链接短于20字符算1字，超出按实际字数算

### 必须包含

- 风险提示（可精简版）："市场有风险"
- 无个股

### 禁止表达

- ❌ "一定涨"、"保证赚钱"
- ❌ 具体个股
- ❌ 极端词汇（"崩盘"、"暴涨"）

## 微博示例

### 示例1：市场评论

```
市场缩量整固，节前大概率维持震荡。成交额跌破万亿，情绪偏谨慎。#A股# #市场分析# #投资干货# 你怎么看？
```

### 示例2：热点解读

```
这个政策的影响，远比表面看起来更深。表面是调整，底层是方向信号。#市场解读# #政策分析# #投顾观点# 你觉得呢？
```

### 示例3：投教营销

```
什么是指数基金？简单说就是一篮子股票，省心又省力。适合没时间研究个股的朋友。#基金入门# #理财干货# #投资有风险# 还有什么问题？评论区见
```
