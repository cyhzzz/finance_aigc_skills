# 知乎长文规范

> 本文档是多平台改写流程 Step 6.4 的详细操作手册

---

## 知乎长文输出要求

### 格式要求

| 要求 | 说明 |
|-----|------|
| 文件格式 | Markdown（.md） |
| 标题层级 | H1 + H2 + H3 完整 |
| 行文风格 | 论证式（观点→论据→论证） |
| 引用格式 | 脚注规范 |
| 互动引导 | 可选 |

---

## 知乎文章结构

```markdown
# {标题}

{导语（可选，100字内）}

## {章节1标题}

{正文：观点+论据}

## {章节2标题}

{正文：观点+论据}

## {结论}

{总结+互动引导}

---

## 参考文献

[1] {来源}：{链接}
[2] {来源}：{链接}
```

---

## 标题规范

### 知乎标题特点

- 信息量大
- 含关键词（利于搜索）
- 可含数字/问号
- 相对正式，不过度标题党

### 标题示例

```
# A股2024年行情展望：核心逻辑与投资机会
# 基金定投真的能赚钱吗？数据说话
# 为什么大多数散户买基金亏钱？
```

---

## 正文规范

### 论证结构

每个章节遵循：

```
{观点句（结论先行）}
↓
{论据1：数据/事实}
↓
{论据2：案例/引用}
↓
{论证：逻辑推导}
↓
{小结（可选）}
```

### 段落规则

- 每段不要太长（≤150字）
- 重点句子可用 `**加粗**`
- 列表使用有序列表（论证过程）
- 引用使用 `>` 块

### 知乎特有表达

```markdown
> 「论点」——{来源}

脚注标记[1]：上标格式
```

---

## 脚注规范

### 脚注格式

```markdown
这是一句有来源的话[1]。

这是一句有来源的话[2]。

---

## 参考文献

[1] {文章标题}：{链接}
[2] {文章标题}：{链接}
```

### 脚注位置

- 文中出现处：右上标 [数字]
- 文章末尾：独立 `## 参考文献` 章节

## 知乎合规要求

### 必须包含

- 风险提示
- 数据来源标注
- 无个股直接推荐

### 禁止表达

- ❌ "买这个一定赚"
- ❌ 具体个股（可以说板块/ETF）
- ❌ 夸张标题党

## 知乎文章示例

```markdown
# 为什么大多数散户买基金亏钱？

买基金亏损，已经成为散户投资者的常态。但亏损的根本原因，往往不是基金本身的问题，而是投资行为的问题。

## 一、数据说话：基金赚钱，基民不赚钱

根据支付宝发布的基民报告，持有基金3年以上的用户，盈利比例超过80%。但持有基金不到3个月的用户，亏损比例超过60%。

**核心问题**：持有时间太短，是亏损的主要原因。

这个数据告诉我们一个简单的事实：基金是长期投资工具，但大多数人在用短炒的方式操作。

## 二、行为金融学：为什么我们拿不住？

行为金融学研究发现，人类大脑对损失的反应强度是对收益的2倍（损失厌恶）。这意味着：

- 亏损100元的痛苦 ≈ 盈利200元的快乐
- 市场下跌时，大概率事件是卖出而非加仓
- 恐慌性卖出，往往卖在最低点

> 「在别人恐惧时贪婪，在别人贪婪时恐惧。」——巴菲特

## 三、解决方案：定投

解决拿不住最简单的方法，是定投。

定投的优势：
1. 自动扣款，不需要择时
2. 成本分摊，降低择时风险
3. 养成习惯，长期持有

---

## 结论

基金亏损的根本原因不是基金本身，而是投资行为。用长期持有的心态投资基金，大概率能获得不错的收益。

**风险提示**：历史收益不代表未来，投资有风险，入市需谨慎。
```
